# Armor Status

A lightweight Minecraft mod that displays your armor durability and status in real time using a clean, unobtrusive HUD.

## Features
- Real-time armor durability display
- Clean and minimal HUD
- Lightweight and performance-friendly
- Ideal for survival and hardcore gameplay

## Installation
1. Download the mod from Modrinth.
2. Place the `.jar` file into your Minecraft `mods` folder.
3. Launch the game with the supported mod loader.

## Compatibility
- Client-side mod
- Requires a supported mod loader (Fabric as specified on Modrinth)
